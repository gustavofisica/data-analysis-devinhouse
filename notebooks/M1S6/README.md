# Module 1 - Week 6 (Clamed Data Insights)

This directory contains the comprehensive Clamed Data Insights project: data cleaning, exploratory analysis, feature engineering, and a predictive machine learning model for invoice value prediction.

## Notebooks

- **clamed_data_insights_project.ipynb**: Main project notebook covering:
  - Data loading and cleaning from healthcare claims dataset
  - Exploratory Data Analysis (EDA) with visualizations
  - Feature engineering and categorical encoding
  - Machine Learning pipeline (RandomForest regressor)
  - Model evaluation and feature importance analysis
  - Interactive prediction function for invoice values

## Data Requirements

Ensure `data/input/xlsx/dataset_clamed.xlsx` is available before running the notebook.

## How to run

Open the notebook in Jupyter and execute cells sequentially. The notebook generates visualizations saved in `data/output/images/`.
