# Module 1 - Week 8 (Schemas and DDL/DML Scripts)

This directory contains SQL scripts for database creation, data manipulation, and normalization exercises.

## Files

### Normalization Exercise
- **normalization_exercise.md**: Complete analysis applying 1NF, 2NF, and 3NF to an order table
- **create_tables.sql**: DDL script creating the 4 normalized tables (CLIENTES, PRODUTOS, PEDIDOS, ITENS_PEDIDO)
- **dml_operations.sql**: DML script with INSERT, UPDATE, and DELETE operations

### School Database System
- **school_database.sql**: Complete database script for DEVinHouseSchool (DDL + DML + DQL)
  - Creates database and 7 related tables
  - Includes sample data for students, teachers, classes, rooms, and subjects
  - Contains UPDATE and DELETE operations
  - Includes 8 complex queries with JOINs, aggregations, and filters
