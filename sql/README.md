# SQL

This directory contains all SQL-related files organized by type and purpose.

## Structure

### modeling/
Entity-Relationship diagrams created with BrModelo
- **M1S7/**: Database models (Berçário, E-commerce, Livraria, School)
- **M1S8/**: Normalized database diagram (Order system in 3NF)

### schemas/
DDL (Data Definition Language) and DML (Data Manipulation Language) scripts
- **M1S8/**: Table creation, data manipulation, and normalization exercises

### queries/
DQL (Data Query Language) - SELECT statements
- **M1S8/**: Complex queries with filters, joins, and aggregations

### migrations/
Database migration scripts (currently empty)

### procedures/
Stored procedures and functions (currently empty)
