# Database Modeling

This directory contains Entity-Relationship diagrams created with BrModelo for database design exercises.

## Subdirectories

- **M1S7/**: Database modeling exercises (Berçário, E-commerce, Livraria, School relationships)
- **M1S8/**: Normalized database model (Order system in 3NF)

## File Format

All modeling files use the `.brM3` extension and require BrModelo software to open and edit.
