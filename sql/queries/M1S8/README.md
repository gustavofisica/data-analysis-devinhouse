# Module 1 - Week 8 (DQL Queries)

This directory contains Data Query Language (SELECT) scripts with filters, ordering, and complex operations.

## Files

- **dql_queries.sql**: Collection of SELECT queries demonstrating:
  - Queries for specific customers
  - Filtering by order value (above 500)
  - Ordering by date
  - Using logical operators (AND, OR, NOT)
  - JOINs across multiple tables
  - Aggregations (COUNT, AVG, SUM)
  - Complex filtering with HAVING

- Modeling: `/sql/modeling/M1S8/`
