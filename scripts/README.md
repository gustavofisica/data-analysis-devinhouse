# Scripts

This directory contains Python scripts for exercises and data processing organized by module and week.

## Subdirectories

- **M1S2/**: Module 1, Week 2 - Basic Python scripts (type conversion, guessing game)
- **M1S3/**: Module 1, Week 3 - Data structures (lists, dictionaries, sets, tuples)
- **M1S4/**: Module 1, Week 4 - File I/O, CSV processing, email validation, modularization

## How to run

Run Python scripts from the repository root using Python 3. Example:

```bash
python scripts/M1S2/guessing_game.py
```

For modularized scripts, see the specific folder README (e.g., `scripts/M1S4/README.md`).
