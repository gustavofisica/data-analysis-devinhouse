# Input Excel Files

This directory contains Excel workbook source data.

## Files

- **dataset_clamed.xlsx**: Healthcare claims dataset - used in Module 1, Week 7 Clamed Data Insights project
- **seeds.xlsx**: Seeds dataset in Excel format (from Udemy Pandas course)

## Usage

Read with `pandas.read_excel()` specifying sheet names when necessary. Do not modify original files.
