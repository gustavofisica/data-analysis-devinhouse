# Data Directory

This directory contains all data files used in the Data Analysis - DEVinHouse course projects, organized into input (source data) and output (generated data) folders by file type.

## Directory Structure

```
data/
├── input/
│   ├── csv/          # Input CSV files
│   ├── xlsx/         # Input Excel files
│   ├── txt/          # Input text files
│   └── images/       # Input images (diagrams, references)
└── output/
    ├── csv/          # Generated CSV files
    ├── xlsx/         # Generated Excel files
    ├── txt/          # Generated text files
    └── images/       # Generated images (diagrams, charts)
```

## Data Sources

### Input Data (`input/`)

#### CSV Files

- **`clientes.csv`**, **`pedidos.csv`**, **`produtos.csv`**

## Usage Notes

- **Input files** should not be modified by scripts - they serve as the original source data
- **Output files** are automatically generated and can be safely regenerated by running the corresponding scripts
- All notebooks and scripts reference these files using relative paths from their respective locations
