# Output Images

This directory contains visualizations and plots generated by Jupyter Notebooks and analysis scripts.

## Files

Generated during notebook execution:
- **conclusao_viz1_homogeneidade_multidimensional.png**: Multidimensional homogeneity analysis
- **conclusao_viz2_correlacoes_variaveis_continuas.png**: Correlation analysis of continuous variables
- **conclusao_viz3_distribuicao_custos_padroes.png**: Cost and pattern distribution
- **conclusao_viz4_padroes_temporais.png**: Temporal patterns analysis
- **conclusao_viz5_dimensoes_clinicas.png**: Clinical dimensions analysis
- **M1S7- Livraria.png**: Database modeling diagram

## Usage

Images are generated by notebook execution (e.g., `notebooks/M1S6/clamed_data_insights_project.ipynb`). They are used in reports and presentations. Regenerate by running the corresponding notebook.
