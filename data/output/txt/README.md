# Output Text Files

This directory contains text files generated by validation and processing scripts.

## Files

- **emails_validos.txt**: Valid emails (generated by `scripts/M1S4/email_validator.py`)
- **emails_invalidos.txt**: Invalid emails (generated by `scripts/M1S4/email_validator.py`)

## Usage

Regenerate by running `python scripts/M1S4/email_validator.py`.
