# Output Data

This directory contains data files generated by scripts and notebooks. Files are organized by type and can be safely regenerated by running the corresponding processing scripts.

## Subdirectories

- **csv/**: Generated CSV files
- **txt/**: Generated text files
- **xlsx/**: Generated Excel files (reserved for future use)
- **images/**: Generated visualizations (PNG, PDF plots from notebooks)

## Usage

Output files are produced by running scripts and notebooks. They can be safely deleted and regenerated. Do not rely on them as permanent dataâ€”treat scripts as the source of truth.
